## 🎯 功能说明

### 1. 图像识别

拍照后，系统会自动：
- 识别照片中的物品
- 获取英文单词名称
- 获取单词定义
- 显示在悬浮提示中

### 2. 语音发音

在悬浮提示窗口中：
- 点击小喇叭图标 🔊
- 系统会使用 OpenAI TTS 播放单词发音
- 播放时图标会变为蓝色

### 3. 使用方式

1. **拍照识别**
   - 点击搜索框旁边的圆形拍照按钮
   - 拍照后自动识别物品
   - 显示英文单词和定义

2. **听发音**
   - 鼠标悬浮在单词图片上
   - 点击悬浮窗口中的小喇叭图标
   - 听到清晰的英文发音

## 🐛 故障排除

### 图像识别不工作

1. 检查 `.env` 文件是否存在
2. 确认 API Key 格式正确（以 `sk-` 开头）
3. 查看浏览器控制台错误信息
4. 确认网络连接正常

### 语音播放不工作

1. 检查浏览器是否允许音频播放
2. 查看浏览器控制台错误信息
3. 确认 API Key 有足够的配额
4. 检查网络连接

### 仍然看到错误

1. 重启开发服务器
2. 清除浏览器缓存
3. 检查 `.env` 文件内容
4. 查看浏览器控制台详细错误
