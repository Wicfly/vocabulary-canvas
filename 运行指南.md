# 🚀 运行指南

## 快速启动

### 1. 安装依赖（首次运行）
```bash
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
```

### 3. 打开浏览器
访问：**http://localhost:5173**

---

## 详细步骤

### 步骤 1：打开终端
在项目目录 `/Users/hope/Desktop/website` 中打开终端

### 步骤 2：安装依赖（如果还没安装）
```bash
npm install
```
> 这只需要在首次运行时执行一次

### 步骤 3：启动开发服务器
```bash
npm run dev
```

您会看到类似这样的输出：
```
  VITE v5.0.0  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

### 步骤 4：打开浏览器
点击终端中的链接，或手动访问：
**http://localhost:5173**

---

## 停止服务器

在运行 `npm run dev` 的终端中，按 `Ctrl + C` 停止服务器

---

## 构建生产版本

如果需要构建用于部署的版本：

```bash
npm run build
```

构建后的文件会在 `dist` 目录中

---

## 常见问题

### ❌ 端口被占用
如果 5173 端口被占用，Vite 会自动使用下一个可用端口（如 5174）

### ❌ 依赖安装失败
尝试：
```bash
rm -rf node_modules package-lock.json
npm install
```

### ❌ 图片不显示
- 检查 `.env` 文件中的 API Key 配置
- 查看浏览器控制台的错误信息
- 如果没有配置 API Key，应用会使用占位图片（这是正常的）
