# 🔄 清除缓存指南

## 问题：刷新后看不到更新

这通常是浏览器缓存导致的。以下是解决方法：

## 方法 1：硬刷新（最简单）

### Mac:
- **Chrome/Edge**: `Cmd + Shift + R` 或 `Cmd + Option + R`
- **Safari**: `Cmd + Option + R`
- **Firefox**: `Cmd + Shift + R`

### Windows:
- **Chrome/Edge**: `Ctrl + Shift + R` 或 `Ctrl + F5`
- **Firefox**: `Ctrl + Shift + R`
- **Safari**: `Ctrl + F5`

## 方法 2：清除浏览器缓存

### Chrome/Edge:
1. 按 `F12` 打开开发者工具
2. 右键点击刷新按钮
3. 选择 "清空缓存并硬性重新加载"

### Safari:
1. 按 `Cmd + Option + E` 清空缓存
2. 然后刷新页面

### Firefox:
1. 按 `Cmd + Shift + Delete` (Mac) 或 `Ctrl + Shift + Delete` (Windows)
2. 选择 "缓存"
3. 点击 "立即清除"

## 方法 3：使用无痕模式

打开浏览器的无痕/隐私模式，访问网站：
- **Chrome**: `Cmd + Shift + N` (Mac) 或 `Ctrl + Shift + N` (Windows)
- **Safari**: `Cmd + Shift + N`
- **Firefox**: `Cmd + Shift + P` (Mac) 或 `Ctrl + Shift + P` (Windows)

## 方法 4：禁用缓存（开发时）

在开发者工具中：
1. 按 `F12` 打开开发者工具
2. 打开 **Network** 标签
3. 勾选 **"Disable cache"**
4. 保持开发者工具打开

## 方法 5：清除特定网站的缓存

### Chrome:
1. 按 `Cmd + Shift + Delete` (Mac) 或 `Ctrl + Shift + Delete` (Windows)
2. 选择 "缓存的图片和文件"
3. 时间范围选择 "过去 1 小时"
4. 点击 "清除数据"

## 验证更新

清除缓存后，你应该能看到：
- ✅ 书本样式的 Gallery 页面
- ✅ 左右分页交互
- ✅ 自定义鼠标光标（椭圆，显示 last page/next page）
- ✅ 左右各 3 个单词的布局

## 如果还是不行

1. **检查开发服务器**：
   ```bash
   # 确保开发服务器正在运行
   npm run dev
   ```

2. **检查控制台错误**：
   - 按 `F12` 打开开发者工具
   - 查看 Console 标签是否有错误

3. **检查网络请求**：
   - 在 Network 标签中
   - 查看文件是否加载成功
   - 检查文件的时间戳是否是最新的

4. **重启开发服务器**：
   ```bash
   # 停止服务器 (Ctrl + C)
   # 然后重新启动
   npm run dev
   ```
